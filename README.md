## 🏫 42 — Curriculum

<details>
<summary><b>✅ Milestone 0 &nbsp;—&nbsp; Libft</b></summary>
<br/>

**Language:** C

Reimplementation of 40+ functions from `libc` from scratch — memory, strings, and linked list API (bonus). The foundation everything else is built on.

<br/>
</details>

<details>
<summary><b>✅ Milestone 1 &nbsp;—&nbsp; ft_printf · get_next_line · push_swap</b></summary>
<br/>

**ft_printf** — Variadic `printf` implementation. Handles `%c %s %d %i %u %x %X %p %%` via format string parsing and `va_list`.

**get_next_line** — Reads any file descriptor one line at a time across calls. Arbitrary `BUFFER_SIZE`, no leaks. Bonus: multiple simultaneous FDs.

**push_swap** — Sorts a stack of integers using only two stacks and 11 operations. Graded by operation count — the algorithm has to be efficient to pass.

<br/>
</details>

<details>
<summary><b>✅ Milestone 2 &nbsp;—&nbsp; Born2beroot · A-Maze-ing · Python Modules</b></summary>
<br/>

**Born2beroot** — Debian VM configured from scratch: encrypted LVM partitions, SSH hardening, UFW, sudo policy, password enforcement, cron monitoring script. Bonus: WordPress stack (Lighttpd + MariaDB + PHP-FPM).

**A-Maze-ing** — Maze generation and pathfinding project. Algorithmic challenge focused on graph traversal and display logic.

**Python Modules** — Progressive series covering Python fundamentals through OOP, file handling, and standard library usage. Each module is a standalone graded project.

<br/>
</details>

<details>
<summary><b>⏳ Exam Rank 02</b></summary>
<br/>

Timed C exam. Solo, no internet. Tests core skills: string manipulation, memory management, and algorithm implementation under strict time constraints.

<br/>
</details>
